# Calculator Project 🧮

This is my first Python project.

I made this calculator using basic Python concepts. It can do different mathematical operations by taking input from the user.

## What it can do

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Power (**)
- Division (/)
- Floor division (//)
- Remainder (%)

## What I learned while making this

- Taking input from users
- Using if, elif and else
- Using while loop
- Working with operators
- Converting input into numbers

## How to run

1. Download the file
2. Open it with Python
3. Run:

python calculator.py

Enter first number: 10

Enter operator: +

Enter second number: 5
